Lead Response Checklist

A concise working checklist for service businesses that want a clearer inquiry path, faster first replies, and a follow-up loop that does not depend on memory.

Request 3-Point Audit

The checklist

Use this before adding more software.

If any step below is unclear, the lead path is probably slower than it looks. Start by making ownership and next actions explicit.

1. Capture

List every place a lead can arrive: form, inbox, phone, booking page, SMS, CRM, or marketplace message.

2. Route

Name one owner for each inquiry type so the first touch does not sit in a shared queue.

3. Reply

Write the first response before the lead arrives, including what to say, who sends it, and what happens next.

4. Follow up

Define the second and third touches now so the thread keeps moving when the buyer goes quiet.

5. Track

Keep one lead log with source, owner, status, next action, and stop condition.

6. Stop

Document when follow-up ends for replies, bookings, opt-outs, or closed deals.

Missing ownership

If two people think they own the same lead, response time usually slips.

No follow-up map

If the sequence ends after one message, the missed opportunity rate stays hidden.

Scattered sources

If inquiries live in multiple tools, a simple audit is usually the fastest next step.

This checklist is a working note, not a promise of results. It is meant to make the inquiry path easier to inspect and improve.
